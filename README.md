# README

Todos application build on Rails 5 + Foundation + jQuery 

  [Live app](https://todos-rails-5-jquery.herokuapp.com/)

Application versions installed: 

* Rails 5.1.5
* Ruby 2.4.1
* Foundation 6.4
* jQuery 

Prepare: 
  - > bundle install
  - > rake db:setup (rake db:create & rake db:migrate)
